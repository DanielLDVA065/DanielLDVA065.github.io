What is HTML?
HTML es un formato que se usa principalmente para hacer páginas web. Referencia: https://developer.mozilla.org/es/docs/Learn_web_development/Getting_started/Your_first_website/Creating_the_content#:~:text=El%20Lenguaje%20de%20Marcado%20de,p%C3%A1gina%20web%20y%20sus%20contenidos.

Head and Body
Head funciona para indicar que es un header, o sea el título o texto más importante y body es para marcar cual es el texto que conforma la página, siendo mucho más contenido que en head. Referencia: https://www.geeksforgeeks.org/html/explain-the-significance-of-head-and-body-tag-in-html

Basic Tags

section, article, aside, header, footer
section = Se usa para la parte más grande de la pagina
article = Marca la parte indepediente de texto en una pagina
aside = Sirve para indicar informacion que es extra
header = Indica que es la parte superior del texto
footer = Funciona para indicar la parte inferior del texto
Reference: https://developer.mozilla.org/en-US/docs/Web/HTML/Element

What is CSS?
CSS funciona para darle formato a la pagina, con esto puede manipular los colores, diseño y tamaño de las letras.
Reference: https://developer.mozilla.org/en-US/docs/Web/CSS

Purpose of CSS
Sirve para darle un formato más bonito a un HTML, sin CSS es más dificl lograr estilos elaborados.
Reference: https://www.w3schools.com/css/

How to apply CSS
1. Inline (Tag desde dentro)
2. Internal (Style desde dentro)
3. External (Archivo separado)
Reference: https://developer.mozilla.org/en-US/docs/Web/CSS/How_to

Selector and Rule
Selector = Es el elemento que eliges
Rule = Es lo que cambias, como el tamaño, color o fuente de la letra
Reference: https://developer.mozilla.org/en-US/docs/Web/CSS/Syntax

Selectors
Tag selector = Sirve para elegir todas las tags, como: p, h1, h2, etc.
Class selector = .name
Tag + Class = p.name
ID selector = #name
Reference: https://developer.mozilla.org/en-US/docs/Web/CSS/CSS_Selectors

Class and ID
Class = Style que se puede volver a usar
ID = Es un elemento unico
Reference: https://www.w3schools.com/css/css_id_class.asp

Box Model
Es cuando todos los elementos tienen contenido, borde, relleno y un margen
Reference: https://developer.mozilla.org/en-US/docs/Web/CSS/box_model

Media Queries
Sirve para adaptar tu HTML a diferentes tipos de pantallas, como a una computadora o un telefono
Reference: https://developer.mozilla.org/en-US/docs/Web/CSS/Media_Queries

display: grid
Hace columnas y filas como una tabla de una forma flexible
Reference: https://developer.mozilla.org/en-US/docs/Web/CSS/grid

display: flex
Funciona para acomodar elementos de forma dinamica en filas o columnas
Reference: https://developer.mozilla.org/en-US/docs/Web/CSS/flex
